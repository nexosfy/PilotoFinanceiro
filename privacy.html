Política de Privacidade

Última atualização: março de 2025

1. Sobre esta política

Esta Política de Privacidade descreve como o PilotoFinanceiro trata os seus dados ao usar o aplicativo. Leia com atenção para entender nossas práticas.

2. Dados que coletamos

O PilotoFinanceiro não coleta, não armazena e não transmite nenhum dado pessoal para servidores externos. O aplicativo funciona 100% offline.

Todos os dados inseridos no app (registros de dias, configurações, cofrinhos, abastecimentos) ficam armazenados exclusivamente no armazenamento local do seu dispositivo.

3. Dados que NÃO coletamos

  • Nome, e-mail ou qualquer dado de identificação pessoal
  • Localização geográfica
  • Dados de uso ou comportamento dentro do app
  • Informações financeiras ou bancárias
  • Dados do dispositivo (modelo, sistema operacional, etc.)

4. Armazenamento local

Os dados ficam salvos no armazenamento interno do seu celular (localStorage). O PilotoFinanceiro não tem acesso a esses dados — eles pertencem somente a você.

Ao usar a função de Backup, o arquivo exportado fica salvo localmente no seu dispositivo. Nenhuma cópia é enviada para servidores externos.

5. Acesso à internet

O PilotoFinanceiro não requer conexão com a internet para funcionar. A única situação em que a internet é usada é durante o download inicial do aplicativo.

A landing page (site) utiliza o Google Analytics para medir o número de visitas e cliques. O Analytics pode coletar dados anônimos de navegação conforme a Política de Privacidade do Google.

6. Compartilhamento de dados

Não compartilhamos nenhum dado com terceiros, pois não coletamos nenhum dado.

7. Segurança

Como os dados ficam apenas no seu dispositivo, a segurança dos seus dados depende da segurança do próprio celular. Recomendamos manter o aparelho com senha ou biometria ativada.

8. Seus direitos

Como não coletamos dados pessoais, não há dados a solicitar, corrigir ou excluir nos nossos sistemas. Para apagar seus dados do app, basta desinstalar o aplicativo do seu dispositivo.

9. Alterações nesta política

Podemos atualizar esta política periodicamente. A data de atualização estará sempre indicada no topo desta página.
